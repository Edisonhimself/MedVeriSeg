## :mega: News
- **2026.6.05**: We’ve uploaded our paper *MedVeriSeg: Teaching LISA-Like Medical Segmentation Models to Verify Query Validity Without Extra Training* to arXiv and set up this repository! Welcome to **watch** 👀 this repository for the latest updates.
- **2026.6.06** We have uploaded the code for **MedVeriSeg**. It includes the code for the **Similarity Response Quality Scoring Module** and the **Lightweight Routed Multi-Agent Verification Module**. You can integrate them into any **LISA-like medical image segmentation model** you would like to use.
